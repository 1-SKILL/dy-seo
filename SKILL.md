## 执行流程

### 第零步:获取最新算法规则

执行前先拉取抖音搜索算法的最新变化:

```bash
curl -s "https://raw.githubusercontent.com/1-SKILL/dy-seo/main/latest-rules.json"
```

### 第一步:了解优化目标

1. **账号定位?** — 行业/领域/人设
2. **核心关键词?** — 想让用户搜什么词找到你
3. **目标用户?** — 年龄/城市/需求
4. **当前问题?** — 搜不到/播放量低/粉丝不涨

### 第二步:关键词研究

| 类型 | 示例 | 流量 | 竞争 |
|------|------|------|------|
| 核心词(1-2字) | "护肤""减肥" | 极大 | 极高 |
| 长尾词(3-5字) | "油皮护肤推荐" | 中等 | 中低 |
| 蓝海词(5+字) | "油皮秋冬面霜推荐2026" | 较小 | 极低 |

**挖词方法:**
1. 抖音搜索框下拉联想词
2. 搜索结果底部"相关搜索"
3. 同行热门视频评论区

输出:至少20个关键词,按优先级排列。

### 第三步:7个优化点

**1.标题:** 核心词放前15字,用疑问句或数字增加点击
**2.描述:** 第一行放核心词,50-100字,自然嵌入长尾词
**3.标签(金字塔):**
```
热门(1-2个):曝光 → #护肤 #好物推荐
垂直(2-3个):匹配 → #油皮护肤 #面霜推荐
长尾(3-5个):排名 → #油皮秋冬面霜 #平价面霜
```
**4.口播关键词:** 口播中提到的词会被索引,自然提及核心词
**5.评论区:** 置顶含关键词的评论,引导互动
**6.合集:** 合集名=核心词,同合集视频权重叠加,每词做5-10个视频
**7.发布时间:** 早7-9,午12-14,晚18-22。发布后1小时互动决定推荐

### 第四步:输出优化方案

```
===== 抖音SEO优化方案 =====

【账号定位】XX领域/XX人设
【关键词清单】核心词/长尾词/蓝海词各5-10个
【标题优化】当前→优化后(嵌入关键词)
【描述模板】[优化后描述]
【标签(金字塔)】[标签列表]
【合集规划】合集名+建议视频主题
【评论区置顶】[互动引导]
【发布建议】最佳时间+前1小时策略
```

## 持续监测

- 每周搜核心词,记录排名变化
- 关注数据面板"搜索流量"占比
- 排名上升但点击低→优化封面标题
- 排名不动→增加同关键词视频数量

## 注意事项

- 不堆砌关键词(会被降权)
- 内容质量是基础,SEO是放大器
- 本地商家重点做地域词
