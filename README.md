# 抖音SEO (dy-seo)

> 抖音搜索优化 · 关键词金字塔 · 搜索流量占比从 3% 提升到 25%。

**dy-seo** 让别人靠运气等推荐，你靠搜索收割精准流量。关键词策略和算法规则每 2-3 天同步更新。

## Install

```bash
npx skills add 1-SKILL/dy-seo -g -y
```

## 解决什么问题

- 每天拍视频全靠算法推荐"赌"流量
- 流量忽高忽低完全没有规律
- 视频在搜索结果里根本找不到

**90% 创作者只做推荐流量，忽略搜索流量——这就是你的机会。**

## 效果

| 指标 | 优化前 | 优化后 |
|------|-------|-------|
| 搜索流量占比 | 3% | 25% |
| 核心词排名 | 搜不到 | 首页 |
| 流量稳定性 | 靠天吃饭 | 持续稳定 |
